JSON Format Guide

Structure

  • collections: Array of collections to create
  • nodes: Array of nodes (only works when inside a collection)
  • subCollections: Nested collections within a parent collection

Note: Standalone nodes at the root level will only be created if you're currently viewing a collection.

Collection Fields

  • name: (Required) Collection name
  • description: (Optional) Collection description
  • nodes: (Optional) Array of nodes within this collection
  • subCollections: (Optional) Nested collections

Node Fields

  • name: (Required) Node name
  • description: (Optional) Node description
  • customFields: (Optional) Array of custom fields

Custom Field Structure

  • fieldName: (Required) Display name of the field
  • fieldType: (Required) Type of the field
  • fieldValue: (Required) Value of the field

Custom Field Types

Supported types: text, number, date, url, imageUrl, richText
